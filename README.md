# @node-fi/node-sdk-react-native

A React Native wrapper for the Node Core SDK

## Installation

```sh
npm install @node-fi/node-sdk-react-native
```

## Usage

```js
import { multiply } from "@node-fi/node-sdk-react-native";

// ...

const result = await multiply(3, 7);
```
